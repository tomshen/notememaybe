Note Me Maybe

Note Me Maybe is a minimalistic online notepad.

  1. Write text in Markdown
  2. Format it with a click of a button

Text saves automatically, but you can also press a button to save manually.

To create a note with a custom URL, go to /n/<custom_url>. If a note doesn't already exist at that URL, a new one will be created.

Note Me Maybe was created by Tom Shen. Source code is on Github.
